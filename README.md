O objetivo da atividade foi aprender quais são e como funcionam as principais tags HTML e seus atributos, para podermos estruturar corretamente uma página HTML.
Fiz um relatório de pesquisa em docs, registrando as principais tags para a estrutura básica de um documento HTML.
Para abrir o código HTML no navegador, selecione o arquivo .html no seu computador, clique com o botão direito e selecione "Abrir com", escolhendo seu navegador de preferência.
